# 📊 ília Dashboards - React + TypeScript + Chart.js

Technical test developed for ília focused on dashboard creation using Chart.js in a React + TypeScript environment.

## 🚀 Stack

- React 19
- TypeScript 5.9
- Chart.js 4
- react-chartjs-2 5
- Vite 8
- React Router 7 Framework mode
- TailwindCSS 4

## ⚙️ Getting Started

### Prerequisites

- **Node.js** >= v24.14.1
- **npm** >= 11.11.0

### Installation

```bash
# Clone the repository
git clone https://github.com/luizfelipeleite/ilia-charts

# Enter the project folder
cd ilia-charts

# Install dependencies
npm install
```

### Running in development

```bash
npm run dev
```

The app will be available at: **http://localhost:5173**

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```
